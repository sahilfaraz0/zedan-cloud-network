# Networking in the Cloud (ZEDAN Enterprises)

**Author:** Sahil Faraz | **Date:** June 2025

---

## 📋 Project Overview

This repository contains the Cisco Packet Tracer network simulations developed for a **Networking in the Cloud** assignment which included *Principles, Implementation, and Performance Enhancement*.

The project simulates a hybrid cloud network designed for **ZEDAN Enterprises, Lahore**, commissioned by **Stratosphere Networks, Islamabad**. It demonstrates the design, implementation, testing, and optimization of a multi-site enterprise cloud network.
